# Snake
A Snake clone built in C++ using the ncurses library!


https://github.com/mikeTwoTimes/snake/assets/135299307/410af842-81c6-40a7-968e-f80f5d6143e8


## Dependencies
- C++ 17
- Ncurses library
- Terminal color
- Minimum terminal dimensions of 68x35
